# ✈️ British Airways Review Analysis Dashboard

## 📌 Project Overview

This project analyzes customer reviews of British Airways using Tableau to uncover insights into passenger satisfaction, service quality, and performance trends.

## 🎯 Objective

* Understand customer satisfaction patterns
* Identify key factors affecting ratings
* Provide insights for business improvement

## 📊 Dashboard Features

* Overall rating KPI
* Ratings by seat type, aircraft, and traveller type
* Time-series analysis of reviews
* Country-wise review distribution (map)
* Detailed service ratings (food, comfort, staff, etc.)

## 🛠️ Tools Used

* Tableau
* Python (for data cleaning, if used)
* CSV datasets

## 📂 Dataset

The dataset contains:

* Customer reviews
* Ratings (overall + service categories)
* Seat type, aircraft type
* Traveller type and country
